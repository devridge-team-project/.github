## 개발자들을 위한 지식 공유 플랫폼, DEVRIDGE 🌁

### 배포 주소
[Devridge](https://devridge-client.vercel.app/)

### 프로젝트 소개
> Bridge for Developer, DEVRIDGE
>
* 개발자들을 위한 정보 공유 플랫폼 웹 사이트입니다.
* 커피챗, Q&A 지식 공유와 같은 개발자들을 위한 다양한 기능들을 제공합니다.
